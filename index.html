    A location based service application

    • detect GPS Location
    • simple check-in feature
    • display location on Maps using Android Maps API

    Aktifkan GPS terlebih dahulu sebelum menggunakan fitur-fitur yang ada pada TWOH's Maps. Karena aplikasi ini hanya menggunakan GPS sebagai penanda lokasi. Apabila GPS sudah aktif, dan koordinat didapatkan semua fitur akan bisa berjalan dengan baik.
